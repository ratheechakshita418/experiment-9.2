# 🚀 React CI/CD Pipeline with GitHub Actions

This project demonstrates a complete CI/CD pipeline using GitHub Actions for a React application.

---

## 📌 Features

- ✅ Automated Testing on Pull Requests
- 🐳 Docker Image Build & Push
- 📦 Deployment to GitHub Container Registry (GHCR)
- 🔔 Slack Notifications on Deployment
- 🏷️ Image tagging with:
  - latest
  - commit SHA

---

## 🛠️ Tech Stack

- React.js
- GitHub Actions
- Docker
- GitHub Container Registry
- Slack Webhooks
